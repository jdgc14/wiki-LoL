## Installation:

1. Clone the repo
```sh
git clone https://github.com/jdgc14/wiki-LoL.git
```
2. Go to directory project
```sh
cd wiki-LoL/
```
3. Install NPM packages
```sh
npm install
```
4. Run project
```sh
npm run dev
```

## Tools:
**React JS ⚛️**, **Redux JS**, and use tools like **React Router**, **Bootstrap**.
